# 📖 Overview

DataMantri AI is an **AI-powered Business Intelligence Platform** that transforms raw business datasets into actionable business intelligence.

Instead of manually analyzing spreadsheets using multiple tools, users simply upload their datasets and DataMantri automatically generates:

- 📊 Business KPIs
- 📈 Interactive Dashboards
- 📉 Trend Analysis
- 🤖 Machine Learning Forecasts
- 🚨 Anomaly Detection
- 💡 AI Recommendations
- 📋 Executive Summaries
- 🎯 Business Health Reports
- 🎙 Voice Business Advisor

The platform combines **Data Engineering**, **Machine Learning**, **Business Intelligence**, and **Full Stack Development** into a single analytics ecosystem.

Rather than implementing all logic inside a single file, the platform follows a service-oriented architecture where each analytics module has a dedicated responsibility.

```text

     ▼
```

# 💾 Smart Caching

Repeated uploads of the same dataset do not trigger full analysis.

The platform computes a SHA-256 hash of the uploaded file and checks previous analyses using:

- Dataset Hash
- Pipeline Version

If a match is found, the cached analysis is returned instantly, reducing computation time and improving user experience.

# 🗄 Database Design

Core database tables include:

```text
users
│
├── upload_history
│       ├── original_filename
│       ├── validation_report
│       ├── result_json
│       └── dataset_hash
│
├── subscriptions
├── feature_usage
├── billing_records
├── payment_records
└── support_tickets
```

The complete analysis is stored inside the `result_json` column, allowing dashboards to be restored without recomputing the analytics pipeline.
